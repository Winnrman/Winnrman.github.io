Everything you need to know about Index.

01

General

What is Index?

Index is a local-first desktop application that makes your files instantly searchable. Add folders, and Index will catalog every file within them — including the content inside PDFs, Word documents, spreadsheets, and more — so you can find anything in seconds.

Is Index free to use?

Index is currently in early access. Join the waitlist to be among the first to try it. Pricing details will be announced closer to the full release.

What platforms does Index support?

Index is available for macOS and Windows. Linux support is planned for a future release.

02

Files & Search

What file types does Index support?

Index supports all major file types including PDF, DOCX, XLSX, PPTX, TXT, Markdown, and more. If a file has readable text content, Index can search inside it.

How many files can I add?

There's no hard limit. Index is designed to handle 10,000+ files without any slowdown. Because everything runs locally, scale is only limited by your machine's storage.

Does Index update automatically when files change?

Yes. Index watches your folders in real time. New files are indexed automatically, changes are reflected immediately, and deleted files are removed from the index. You never need to re-add a folder manually.

Can I add multiple folders from different drives?

Yes. You can add as many folders as you like, across any mounted drives. All of them are unified into a single searchable index.

03

Privacy & Security

Does Index send my files to the cloud?

Never. Index is entirely local. Your files, their contents, and the search index all stay on your machine. Nothing is ever uploaded or transmitted anywhere.

Where is my index data stored?

The index database is stored locally in your application data directory. You have full control over it and can delete it at any time from within the app.
